## 打字效果

```css
h1 {
    width: 0;
    overflow: hidden;
    height: 32px;
    animation: 3s typing steps(9) forwards;
    position: relative;
}

h1::after {
    content: "";
    display: inline-block;
    position: absolute;
    width: 0;
    height: 32px;
    background-color: black;
    right: 0;
    margin-left: 10px;
    animation: 1s cursor steps(2) infinite;
}

@keyframes cursor {
    from {
        opacity: 0;
    }

    to {
        opacity: 1;
    }
}


@keyframes typing {
    from {
        width: 0;
    }

    to {
        width: 16ch;
    }
}
```

```html
<h1>不是因为寂寞才想你</h1>
```
