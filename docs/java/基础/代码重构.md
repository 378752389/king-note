## 类名/方法名/字段重命名

`shift + F6`

## 提取方法/内联方法

* 提取方法: 将公共代码抽出来为一个方法
  `Ctrl + Alt + M`
* 内联方法：将方法内的代码融入到另外一个方法中
  `Ctrl + Alt + N`
  
## 重构变量/重构常量/重构成员变量

* 提取变量： `Ctrl + Alt + C`
* 提取常量： `Ctrl + Alt + V`
* 提取成员变量：`Ctrl + Alt + F`


## 查看字段有在哪些地方被引用

* `Alt + F7`
