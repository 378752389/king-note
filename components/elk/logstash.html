    # 文件配置

    注意点：

    • *FileWatch* 只支持文件的绝对路径，而且会不自动递归目录。使用用数组方式都写明具体哪些文件。
    • 文件路径 path => "/path/to/%{+yyyy/MM/dd/hh}.log"写法，只能写成/path/to/**/*.log，用 ** 来缩写表示递归全部子目录。
    input {
    file {
        path => ["/var/log/*.log", "/var/log/message"]
        type => "system"
        start_position => "beginning"
    }
}
